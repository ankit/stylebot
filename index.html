Stylebot is a browser extension which lets you modify the appearance of any webpage and sync your changes across browsers.

Stylebot for ChromeStylebot for FirefoxStylebot for Edge

Pick and hide ads, change colors, fonts, and lots more.

Launch Stylebot and pick any element to style on a page. Use the Basic editor to quickly apply and save changes to the current website.

Code your own CSS.

Use the Code editor to write your own custom CSS for the page.

Enable readability for articles.

If enabled for a site, Stylebot will automatically hide clutter on all articles for readability.

Pick between Light, Dark and Sepia themes. Use any custom font available on Google Fonts.

Sync your styles across browsers.

Enable sync via Google Drive and keep your styles across profiles and browsers in sync and backed up.

Or download your styles locally.

Stylebot is an open source and free browser extension. Trusted by more than 200,000 users.
